## Getting Started

### Prerequisites
- Node.js
- npm
- MongoDB

### Install
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Test
```bash
npm test
```

### Lint
```bash
npm run lint
```

## Environment Variables
| Variable Name | Description                  |
|---------------|------------------------------|
| `DB_URL`     | Database connection string   |
| `PORT`       | Port to run the server on    |
| `SECRET`     | JWT Secret Key               |
